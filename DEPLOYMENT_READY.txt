SYSTEM OVERVIEW
================================================================================

The Planning Intelligence System is an intelligent data extraction and analysis
agent for dynamic IBP (Integrated Business Planning) API queries. It processes
natural language questions and provides data-driven insights using Azure OpenAI
and real SAP data.

KEY CAPABILITIES:
  ✅ Natural language query processing
  ✅ Intelligent intent classification (5 types)
  ✅ Dynamic entity extraction
  ✅ Scoped metrics computation
  ✅ Azure OpenAI integration
  ✅ Real data validation
  ✅ Hallucination detection
  ✅ Structured response generation

================================================================================
SUPPORTED QUERY TYPES
================================================================================

1. ROOT CAUSE ANALYSIS
   "Why is LOC001 risky?"
   → Identifies risk factors and primary drivers

2. COMPARISON ANALYSIS
   "Compare LOC001 vs LOC002"
   → Compares entities and risk profiles

3. STABILITY ANALYSIS
   "Why is LOC002 not risky?"
   → Explains stability factors

4. TRACEABILITY
   "Show top contributing records"
   → Lists contributing records with metrics

5. SUMMARY
   "What's the planning status?"
   → Provides overall health overview

See SUPPORTED_QUERIES.md for detailed examples and QUICK_QUERY_REFERENCE.txt
for quick reference.

        ↓

├─ Question Classification (5 types)
├─ Entity Extraction (Location, Supplier, Material)

        ↓

├─ Metrics Calculation
└─ Context Building
        ↓
