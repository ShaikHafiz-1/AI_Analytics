5 SUPPORTED QUERY TYPES
================================================================================

1. ROOT CAUSE - "Why is [ENTITY] risky?"
   ├─ Examples:
   │  ├─ "Why is LOC001 risky?"
   │  ├─ "Why is SUP001 having issues?"
   │  └─ "Why is UPS material group risky?"
   ├─ Response: Risk analysis with drivers and metrics
   └─ Metrics: Change rate, primary driver, contribution breakdown

2. COMPARISON - "Compare [ENTITY1] vs [ENTITY2]"
   ├─ Examples:
   │  ├─ "Compare LOC001 vs LOC002"
   │  ├─ "Compare SUP001 vs SUP002"
   │  └─ "Compare UPS vs PUMP"
   ├─ Response: Side-by-side comparison with metrics
   └─ Metrics: Change rates, drivers, risk assessment

3. WHY-NOT - "Why is [ENTITY] not risky?"
   ├─ Examples:
   │  ├─ "Why is LOC002 not risky?"
   │  ├─ "Why is SUP002 stable?"
   │  └─ "Why hasn't LOC001 changed?"
   ├─ Response: Stability analysis
   └─ Metrics: Zero changes, stability factors

4. TRACEABILITY - "Show [RECORDS]"
   ├─ Examples:
   │  ├─ "Show top contributing records"
   │  ├─ "Which records have changed?"
   │  └─ "Show me the risky records"
   ├─ Response: List of contributing records
   └─ Metrics: Record details, deltas, locations

5. SUMMARY - "What's the [STATUS/OVERVIEW]?"
   ├─ Examples:
   │  ├─ "What's the planning status?"
   │  ├─ "How many records changed?"
   │  └─ "What's the forecast delta?"
   ├─ Response: Overall planning health
   └─ Metrics: Global metrics, change rate, health status

================================================================================
KEY METRICS
================================================================================

CHANGE RATE:
  - Percentage of records that changed
  - Formula: (changed_count / total_count) * 100
  - Example: 2 out of 3 = 66.7%

PRIMARY DRIVER:
  - Main reason for changes
  - Options: quantity, supplier, design, schedule
  - Determined by: which affects most records

CONTRIBUTION BREAKDOWN:
  - Percentage of each change type
  - Example: quantity: 50%, supplier: 50%

TOP CONTRIBUTING RECORDS:
  - Records with largest deltas
  - Sorted by absolute delta value
  - Limited to top 5

FORECAST DELTA:
  - Total change in forecast quantity
  - Aggregated across all records
