================================================================================
PROBLEM STATEMENT
================================================================================

The Copilot UI is sending natural language questions, but the backend is NOT
processing them through the NLP/LLM layer. Instead, it returns default responses.

SYMPTOMS:
  ❌ Same response repeated multiple times
  ❌ Follow-up questions ignored
  ❌ Out-of-scope questions return wrong responses
  ❌ No intent routing or entity extraction

ROOT CAUSE:
  • function_app.py expects structured query_type parameter
  • Copilot sends natural language questions
  • No NLP layer to convert questions → query_type
  • Questions bypass the NLP pipeline

================================================================================
SOLUTION OVERVIEW
================================================================================

Created a complete NLP/LLM integration layer that:

✅ Processes natural language questions
✅ Classifies intent (5 types)
✅ Extracts entities (location, supplier, material)
✅ Routes to appropriate handlers
✅ Generates correct responses
✅ Supports multi-turn conversations
✅ Handles out-of-scope questions
✅ Maintains conversation history

SUPPORTED QUERY TYPES
================================================================================

1. SUMMARY
   Pattern: "What's the planning status?"
   Response: Overall metrics and drivers
   Example: "📊 Planning Intelligence Summary: 13,148 total records..."

2. ROOT CAUSE
   Pattern: "Why is LOC001 risky?"
   Response: Risk analysis for specific entity
   Example: "⚠️ Risk Analysis for LOC001: 2 of 3 records changed..."

3. COMPARISON
   Pattern: "Compare LOC001 vs LOC002"
   Response: Side-by-side comparison
   Example: "📊 Comparing LOC001 vs LOC002: LOC001 has more changes..."

4. WHY-NOT
   Pattern: "Why is LOC002 not risky?"
   Response: Stability analysis
   Example: "✅ Stability Analysis for LOC002: 0 records changed..."

5. TRACEABILITY
   Pattern: "Show top contributing records"
   Response: List of top records
   Example: "📊 Top Contributing Records: 1. LOC001 - MAT001..."

6. OUT-OF-SCOPE
   Pattern: "What is your name?"
   Response: Clarification about capabilities
   Example: "I'm a Planning Intelligence assistant..."
