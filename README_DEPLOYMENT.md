## 🚀 7-Step Deployment

### Step 1: Move Files (5 min)
Copy files from current location to org laptop using `QUICK_MOVE_CHECKLIST.md`

### Step 2: Install Dependencies (5 min)
```bash
cd planning_intelligence
pip install -r requirements.txt

cd ../frontend
npm install
```

### Step 3: Test Blob Connection (2 min)
```bash
cd planning_intelligence
python test_blob_real_data.py
```

### Step 4: Load Real Data (2 min)
```bash
python run_daily_refresh.py
```

### Step 5: Deploy to Azure Functions (10 min)
```bash
az login
az functionapp config appsettings set --name <app> --resource-group <rg> --settings BLOB_CONNECTION_STRING="..."
func azure functionapp publish <app>
```

### Step 6: Deploy Frontend (5 min)
```bash
cd frontend
npm run build
# Upload build/ folder to hosting
```

### Step 7: Verify (5 min)
- Dashboard loads with real data
- Planning health shows 37/100
- Copilot responds to questions

## 🔍 Troubleshooting

### Blob Connection Fails
- Check `local.settings.json` has `BLOB_CONNECTION_STRING`
- Verify Azure Storage account is accessible
- Verify files exist in blob storage

### Frontend Can't Reach Backend
- Check `frontend/.env` has `REACT_APP_API_URL`
- Verify URL points to your Azure Functions
- Check CORS settings

### npm install Fails
```bash
npm cache clean --force
rm -rf node_modules
npm install
```

### Python Dependencies Fail
```bash
pip install --upgrade pip
pip install -r requirements.txt
```
