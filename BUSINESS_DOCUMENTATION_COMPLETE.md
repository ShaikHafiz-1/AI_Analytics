## 🏗️ System Overview

### What It Does
The Planning Intelligence Copilot is an AI-powered assistant that helps your supply chain team make faster, smarter decisions by analyzing real-time planning data through natural language conversations.

### How It Works
1. User asks question in plain English
2. Backend classifies question type
3. Loads planning data from Azure Blob Storage
4. Sends to ChatGPT with business rules
5. ChatGPT generates intelligent response
6. User sees answer with insights

### Key Features
- ✅ 12 types of questions answered
- ✅ 13,000+ records analyzed instantly
- ✅ ChatGPT integration with business rules
- ✅ Real-time insights with drill-down
- ✅ Daily data refresh from SAP
- ✅ Enterprise security (SFI compliant)
