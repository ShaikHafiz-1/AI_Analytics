## Key Concepts

### detailRecords
- **What**: Array of 13,148 records with planning data
- **Where**: Created by backend's daily-refresh() function
- **How**: Sent to frontend in DashboardResponse
- **Why**: Provides full context for ChatGPT analysis

### DashboardResponse
- **What**: Response from backend's daily-refresh() endpoint
- **Contains**: Metrics, summaries, and detailRecords
- **Sent to**: Frontend
- **Used by**: DashboardPage to build DashboardContext

### DashboardContext
- **What**: Context object passed to CopilotPanel
- **Contains**: Extracted fields from DashboardResponse
- **Includes**: detailRecords
- **Used by**: CopilotPanel to send with user questions

### The Fix
- **What**: Two critical fixes applied
- **Fix 1**: Timeout increased (6s → 35s)
- **Fix 2**: detailRecords passed in context
- **Impact**: 1-2 seconds faster response, no premature timeouts
