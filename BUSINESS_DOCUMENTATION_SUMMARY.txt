SYSTEM OVERVIEW
================================================================================

WHAT IT DOES
The Planning Intelligence Copilot is an AI-powered assistant that helps your
supply chain team make faster, smarter decisions by analyzing real-time planning
data through natural language conversations.

HOW IT WORKS
1. User asks question in plain English
2. Backend classifies question type
3. Loads planning data from Azure Blob Storage
4. Sends to ChatGPT with business rules
5. ChatGPT generates intelligent response
6. User sees answer with insights
